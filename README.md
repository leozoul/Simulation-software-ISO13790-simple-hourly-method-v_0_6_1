# Simulation-software-ISO13790-simple-hourly-method-v_0_6_1
This repository contains a building enery simulation software based on a modified version ISO13790/2017-simple hourly method.

Overview
The software is consisted of two main scripts that synergize with each other:
  The building structure and calculation procedure according to ISO13790-simple hourly method
  The software UI that allows a user to structure a building, import climate file and configure user preferences. Then dynamic daily energy simulations can be performed to calculate the total energy demand of the simulated building.
  
Features
  The program considers a building as a single lumped thermal zone surrounded by the external environment
  Currently, it is able to consistently simulate Residential and Office buildings.
  Buildings can be saved and loaded from .xlsx (Excel) with specific format
